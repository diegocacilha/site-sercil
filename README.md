# Sercil - Site Institucional

![Next.js](https://img.shields.io/badge/Next.js-16.2.3-black?logo=next.js)
![React](https://img.shields.io/badge/React-19.2.4-blue?logo=react)
![TypeScript](https://img.shields.io/badge/TypeScript-5.0-blue?logo=typescript)
![Tailwind](https://img.shields.io/badge/Tailwind-4.0-06B6D4?logo=tailwindcss)

Site institucional da **Sercil**, uma software house especializada no desenvolvimento de sistemas web personalizados. O site apresenta os serviços da empresa, portfólio de projetos e um formulário para captação de leads.

🌐 **Acesse:** [sercil.com.br](https://sercil.com.br)

---

## ✨ Funcionalidades

- **Design responsivo e minimalista** — Adaptado para todos os dispositivos
- **Otimizado para SEO** — Metadados e estrutura semântica para mecanismos de busca
- **Formulário de contato** — Captura de e-mails de potenciais clientes
- **Seção de portfólio** — Vitrine para demonstração de projetos
- **Performance** — Construído com Next.js para carregamento rápido
